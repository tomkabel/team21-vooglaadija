## 🚀 Quick Start

### Before Sprint 1

1. Read [sprint-1-foundation.md](./sprint-1-foundation.md)
2. Review [sprint-backlog-overview.md](./sprint-backlog-overview.md)
3. Run planning meeting using [sprint-1-planning-agenda.md](./templates/sprint-1-planning-agenda.md)

### During Sprint

1. Use [daily-standup-template.md](./templates/daily-standup-template.md) for standups
2. Track progress on GitHub Project board
3. Update [sprint-1-burndown.md](./tracking/sprint-1-burndown.md) daily

### After Sprint

1. Conduct review using [sprint-review-template.md](./templates/sprint-review-template.md)
2. Conduct retrospective using [sprint-retrospective-template.md](./templates/sprint-retrospective-template.md)
3. Update [velocity-tracker.md](./tracking/velocity-tracker.md)
4. Plan next sprint
